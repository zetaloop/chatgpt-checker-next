# ChatGPT Checker Next

获取 ChatGPT 和 Grok 的功能、服务等信息。

## 安装

1. 首先需要安装 [Tampermonkey](https://www.tampermonkey.net)
2. 然后点击链接安装此脚本 [chatgpt-checker-next.user.js](https://github.com/zetaloop/chatgpt-checker-next/raw/refs/heads/main/chatgpt-checker-next.user.js)
3. 打开 ChatGPT / Codex / Sora / Grok，页面右侧有一个圆圈，将鼠标光标放上去即可查看模型、额度、服务质量等信息。

## 功能

#### ChatGPT
- **服务质量**：网站的 Proof Of Work 挑战难度值，数值越大通常代表风控越轻，但并不是唯一的判断标准。
- **默认模型**：用户选择的默认模型。
- **价格地区**：当前购买会员的价格地区。
- **模型记忆**：记忆已用/总共的 Token 容量。
- **深度研究**：Deep Research 模式的剩余次数与重置时间。
- **代理模式**：Agent 模式的剩余次数与重置时间。
- **文件上传**：文件上传的剩余次数与重置时间。
- **Codex 用量**：Codex 的每五小时/每周/代码审查用量进度条与重置时间。
- **Sora 模型**：当前可用的 Sora 模型，例如 `turbo` 为 Sora 1，`sy_8` 为 Sora 2。
- **Sora 次数**：Sora 的每日次数与重置时间（Sora 2）、并行数量（Sora 1）。
- **积分**：单独购买的 ChatGPT 积分额度，可用于 Sora、Codex 等任务。
#### Grok
- **订阅**：当前激活的 Grok 和 X 会员。
- **账号地区**：当前账号地区。
- **可用模型**：当前可用模型列表。
- **任务**：Grok 定时任务的额度，包括高频（每日）和低频（其他周期）。
- **开发工具**：启用内部调试工具，可以自定义工具、模型、提示词等。
- **其他小功能**：启用抢鲜体验模型、假装解锁不可用的模型、假装自己是会员等。
